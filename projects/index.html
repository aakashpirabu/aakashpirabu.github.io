CARROTSH


carrotsh is a lightweight and secure remote access server that uses the websocket protocol. It provides a simple and accessible method of connecting to servers, as only a browser is required on the client side.

Languages:

  • Python
  • Javascript
  • HTML

Environments:

  • Node.js

Problems encountered:

  • The console would look weird when the client uses any sort of dark mode filters or extensions. This was solved by trial and error, trying different theme combinations and checking them with common dark mode filters like Dark Reader and Chromium's built-in filter.




KBUILDER



kbuilder is a bash script that automates the building, packaging and installing of custom kernels in Arch Linux. I wrote this for my own convenience, but also published it as a public package in the Arch User Repository in case someone else finds it useful.

Languages:

  • Bash

Problems encountered:

  • Updates to the official PKGBUILD on the Arch repositories occasionally made this script broken. This was solved by writing another script that checks the file everyday, updates the sed injection string to work with newer versions, automatically publishes it to the Arch User Repository and updates the source on the GitHub page.



DUCKDUCKAPI
 ❮ ❯ 
duckduckgo-api.herokuapp.com

DuckDuckAPI is a free web scraping api for DuckDuckGo Search. The api allows you to fetch search results in JSON format, which can then be used by other programs or bots for various purposes. It has an easy interface, and is well documented. This project is closed source.

Languages:

  • Python
  • Javascript
  • Rust

Environments:

  • Node.js

Problems encountered:

  • The original backend scraper that was being used no longer worked due to updates to the DuckDuckGo search engine. This required a complete rewrite of the scraping rules, and the issue took about two days to hotfix.



CAT-DATASET



annikav9.github.io/cat-dataset

A dataset of 29843 cat pictures (64x64), compiled together for training models. I created this when I was experimenting with image GANs, and compiled cat images from various sources to train the GANs.

Languages:

  • HTML/CSS
  • Javascript

Problems encountered:

  • Images were to large at first for proper use, and the website hosted to preview them could not handle the bandwidth. So I reduced the file size dramatically, while still ensuring enough image quality for proper training, and moved to GitHub Pages for better hosting performance.
